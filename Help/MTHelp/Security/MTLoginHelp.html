Mother Teres@ Login Help

Figure 1: Sample screen capture of Login screen depicting relevant items.

  1. Please make sure that the database is up and running before logging in.
  2. Type in your given password in the Login Dialogue box. (Password is case sensitive).
  3. Password Typed is shown as ‘XXXX’ for security reasons.
  4. You have 4 chances to type in the correct password.
  5. If Login is successful the ‘Login’ status box will show who is logged in.
  6. If the password is forgotten then you will have to contact your supplier as a new one will have to be generated for you.
  7. Please keep your password secret.
  8. More than one password/person can unlock the system.
  9. If the database is down a message will inform you to that effect as soon as you try to Log In.
  10. You can complete the Login procedure by pressing the ‘Enter’ key or pressing the ‘OK’ button in the Login Dialogue box.
