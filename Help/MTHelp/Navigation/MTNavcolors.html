Mother Teres@ Navigation Colours

Sample screen capture depicting menu colour coding.

The Mother Teres@ menu system is colour coded to simplify the GUI interface for the user. The menu system is colour coded according to the following rules:

  1. Dark Blue: Menu item already completed.
  2. Red: Current active view.
  3. Green: The next menu action available.
  4. Grey: The menu item has not been completed yet.
